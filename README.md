# Joel Jonassi

I am Computer Systems Engineer. I build things that sit at the intersection of hardware and software, where a wrong byte crashes a device, not just a browser tab.

My focus is on closing the gap between people, machines, and the software that connects them. Whether that's a kernel module talking to hardware, a GSM-based ride-sharing system in a region with no internet, or an AI pipeline making sense of agricultural data, I care about solutions that actually work in the real world, not just in ideal conditions.

---

**Things I'm working on / thinking about**

- Embedded systems & kernel-level programming
- RAG pipelines and AI Agents
- ML / DL applied to real problems
- Ride-sharing systems, combining standard app-based flows with a GSM fallback for low-bandwidth areas
- Agriculture × technology
- Building things that shouldn't exist yet
